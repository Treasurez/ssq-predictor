## 分析工作流程

### 第一步：获取历史数据

使用脚本进行增量更新：

```bash
python scripts/fetch_ssq_data.py
```

**数据更新机制**：
- 脚本自动获取 2003 年至今的全部历史数据
- 增量更新：自动合并新数据，去重保存
- 历史数据永久保存，只增不减
- 数据文件：`ssq_history.json`（主数据）、`ssq_analysis.json`（统计分析）

### 第二步：数据分析维度

对历史数据进行多维度分析：

#### 1. 概率统计分析

- **号码频率**：统计各号码出现次数，计算热号和冷号
- **遗漏值分析**：每个号码距上次出现的期数
- **奇偶比分析**：红球奇偶数量分布规律
- **大小比分析**：红球大小号（以17为界）分布
- **区间分布**：三区（01-11、12-22、23-33）号码分布
- **和值分析**：红球和值的区间规律
- **跨度分析**：最大号与最小号的差值规律

#### 2. 走势规律分析

- **连号分析**：连续号码出现的频率和位置
- **重号分析**：与上期相同号码的规律
- **邻号分析**：上期号码±1的号码出现规律
- **斜连号分析**：斜线方向的号码趋势
- **AC值分析**：号码复杂度指标

#### 3. 易经玄学分析（个人化）

**必须询问用户出生日期和随机数字**：
> 请告诉我您的出生年月日（如：1990年5月15日），并随机说3个1-33之间的数字，我将为您进行专属玄学分析。
>
> 基于出生日期，我会计算：
> - 您的生肖与五行属性
> - 个人命数（生命密码）
> - 八字日柱与喜用神
> - 专属幸运数字
>
> 随机数字用于：
> - 梅花易数起卦（主要用途）
> - 与命理数字交叉验证
> - 若与命理高度契合，可选1个作为候选（不宜过多）

**注意**：出生日期和随机数字仅用于当次分析，不做任何存储。

- **八字日柱**：根据出生日期推算天干地支
- **个人命数**：出生日期数字相加至个位数
- **生肖五行**：出生年份对应生肖及五行
- **喜用神选号**：根据八字五行喜忌选号
- **个人化起卦**：用出生日期起卦
- **河图洛书**：数字的先天后天方位

#### 4. 物理学角度

- **熵增原理**：系统向无序发展的趋势
- **随机漫步**：号码变化的布朗运动特征
- **吸引子理论**：号码分布的稳定区域

### 第三步：综合研判

结合以上分析，给出：

1. **主推号码**（胆码）：最有把握的2-3个红球
2. **次选号码**（配码）：概率较高的5-8个红球
3. **防冷号码**：可能回补的冷门号码
4. **蓝球推荐**：主推1个 + 备选2个
5. **推荐组合**：生成3-5注完整号码
