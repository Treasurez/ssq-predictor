# 双色球预测大师 (SSQ Predictor)

中国福利彩票双色球号码预测分析技能，融合数理统计与传统智慧。

## 功能特点

- **个人化分析**：基于出生日期推算生肖、命数、八字、喜用神
- **多维度分析**：概率统计、物理学视角、易经玄学三位一体
- **历史数据**：自动获取 2003 年至今全部开奖数据（3400+ 期）
- **智能推荐**：热号冷号、重号邻号、五行干支综合研判
- **数据智能更新**：自动检测是否需要更新，带重试机制

## 使用方法

### 1. 更新数据

```bash
cd ~/.claude/skills/ssq-predictor
python3 scripts/fetch_ssq_data.py
```

脚本特性：
- 自动检测数据是否最新，无需重复获取
- 备份现有数据到 `ssq_history.json.bak`
- 获取 2003 年至今全部历史数据
- 验证数据有效性（红球 01-33，蓝球 01-16）
- 网络请求带重试机制（3次重试，递增延迟）
- 年份范围自动计算，无需手动维护

### 2. 预测分析

在 Claude 中使用以下触发词：

- "双色球"
- "彩票预测"
- "选号"
- "下期号码"
- "红球蓝球"

### 3. 个人化分析

预测时需提供：
- **出生年月日**（如：1990年5月15日）
- **随机3个数字**（1-33之间）

基于这些信息计算：
- 生肖与五行属性
- 个人命数（生命密码）
- 八字日柱与喜用神
- 梅花易数起卦（随机数主要用于起卦分析）

### 4. 输出报告

分析报告包含：

- 上期回顾（奇偶比、大小比、三区比、和值）
- 数理分析（热号、冷号、重号、邻号）
- 物理学分析（熵增、涨落、吸引子效应）
- 易经研判：
  - **天干地支**：当日干支、个人日柱、六合三合吉数
  - **河图洛书**：五行生成数配对、洛书九宫方位
  - **梅花易数**：本卦变卦、卦象五行吉数
- 综合推荐（胆码、配码、防冷号、蓝球）
- 5 注推荐号码

## 数据来源

历史数据来自中国福利彩票官方图表网站 (tubiao.zhcw.com)。
