# 概率论与统计学基础

## 一、双色球中奖概率

### 基础计算

红球：从33个中选6个 = C(33,6) = 1,107,568 种
蓝球：从16个中选1个 = 16 种
总组合：1,107,568 × 16 = 17,721,088 种

### 各奖项概率

| 奖项 | 中奖条件 | 概率 | 约等于 |
|------|---------|------|--------|
| 一等奖 | 6红+1蓝 | 1/17,721,088 | 1772万分之一 |
| 二等奖 | 6红+0蓝 | 15/17,721,088 | 118万分之一 |
| 三等奖 | 5红+1蓝 | 162/17,721,088 | 10.9万分之一 |
| 四等奖 | 5红+0蓝 / 4红+1蓝 | 7695/17,721,088 | 2303分之一 |
| 五等奖 | 4红+0蓝 / 3红+1蓝 | 137475/17,721,088 | 129分之一 |
| 六等奖 | 2红+1蓝 / 1红+1蓝 / 0红+1蓝 | 1043640/17,721,088 | 17分之一 |

### 期望收益

假设：一等奖500万，投注2元

期望收益 ≈ 0.49元（返奖率约49%）

**结论**：从纯数学角度，彩票是负期望游戏。

## 二、核心概率原理

### 1. 独立事件

每期开奖是**独立事件**，前期结果不影响后期。

常见误区：
- ❌ "某号码很久没出，下期一定出"
- ❌ "刚出的号码不会连续出"
- ✅ 每个号码每期概率相同

### 2. 大数定律

样本量足够大时，频率趋近于概率。

应用：
- 红球理论频率：6/33 ≈ 18.2%
- 实际统计应接近此值
- 短期波动不代表规律

### 3. 赌徒谬误

错误认为随机事件会"自我修正"。

示例：连续10期没出的号码，第11期出现概率仍是6/33，不会更高。

### 4. 回归均值

极端值后倾向于回归平均水平。

应用：
- 极热号码可能降温
- 极冷号码可能回补
- 但具体哪期无法预测

## 三、统计分析方法

### 1. 频率分析

统计各号码历史出现次数。

```
频率 = 出现次数 / 总期数
理论频率（红球）= 6/33 ≈ 0.182
```

偏差分析：
- 正偏差：出现频率 > 理论频率（热号）
- 负偏差：出现频率 < 理论频率（冷号）

### 2. 遗漏值分析

当前遗漏值 vs 历史最大遗漏值

遗漏比 = 当前遗漏 / 历史最大遗漏

| 遗漏比 | 状态 |
|-------|------|
| <0.3 | 正常 |
| 0.3-0.6 | 偏冷 |
| 0.6-0.8 | 深冷 |
| >0.8 | 极度深冷，可能回补 |

### 3. 移动平均

计算最近N期的号码特征平均值。

```
和值MA10 = 最近10期和值之和 / 10
```

用于判断趋势：
- MA上升：和值趋势向大
- MA下降：和值趋势向小

### 4. 标准差

衡量数据离散程度。

```
σ = √(Σ(xi - μ)² / n)
```

应用：
- 判断号码分布是否异常
- 识别极端开奖结果

## 四、概率分布

### 1. 二项分布

某号码在N期中出现k次的概率。

```
P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
p = 6/33（红球）或 1/16（蓝球）
```

### 2. 泊松分布

稀有事件（如连续多期不出）的分布。

```
P(X=k) = (λ^k × e^(-λ)) / k!
λ = 平均发生次数
```

### 3. 正态分布

和值、跨度等指标近似服从正态分布。

```
μ = 均值
σ = 标准差
68%数据在 μ±σ 范围内
95%数据在 μ±2σ 范围内
```

## 五、假设检验

### 检验号码是否"有规律"

原假设H0：开奖完全随机
备择假设H1：存在某种规律

步骤：
1. 收集大量数据
2. 计算检验统计量（如卡方值）
3. 与临界值比较
4. 得出结论

**已知结论**：多项研究表明，双色球开奖符合随机分布，无法证明存在可利用的规律。

## 六、投注策略数学

### 1. 凯利公式

理论最优投注比例。

```
f = (bp - q) / b
b = 赔率
p = 获胜概率
q = 1 - p
```

彩票应用：由于期望为负，凯利公式建议投注比例为0。

### 2. 复式投注概率

红球复式（选N个）：
```
中6个红球概率 = C(6,6) × C(N-6,0) / C(N,6)
```

胆拖投注：设D个胆码全中概率 × 拖码中(6-D)个概率

### 3. 期望值计算

```
E = Σ(奖金i × 概率i) - 投注成本
```

只有E > 0时，从数学角度才值得投注。
