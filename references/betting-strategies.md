# 投注策略与资金管理

## 一、投注类型

### 1. 单式投注

选6个红球 + 1个蓝球，2元/注。

优点：成本最低
缺点：中奖难度最高

### 2. 复式投注

#### 红球复式

| 红球个数 | 注数 | 金额 |
|---------|------|------|
| 7 | 7 | 14元 |
| 8 | 28 | 56元 |
| 9 | 84 | 168元 |
| 10 | 210 | 420元 |
| 11 | 462 | 924元 |
| 12 | 924 | 1848元 |

#### 蓝球复式

| 蓝球个数 | 倍数 |
|---------|------|
| 2 | ×2 |
| 3 | ×3 |
| 4 | ×4 |
| ... | ... |
| 16 | ×16 |

#### 双色复式

红球N个 + 蓝球M个 = C(N,6) × M 注

### 3. 胆拖投注

- 胆码：必须出现的号码（最多5个）
- 拖码：至少出1个的号码

注数 = C(拖码数, 6-胆码数)

示例：2胆5拖
注数 = C(5,4) = 5注 = 10元

优势：成本可控，胆码命中率高时收益大

## 二、选号策略

### 策略1：均衡覆盖

原则：号码分布均匀

```
三区各选2个
大小比 3:3
奇偶比 3:3
```

优点：覆盖面广
缺点：与极端结果无缘

### 策略2：热号追踪

选择近30期出现频率最高的号码。

依据：热号可能继续热

风险：热号随时可能转冷

### 策略3：冷号守候

选择遗漏值较高的号码长期追号。

依据：回归均值

风险：冷号可能继续冷，资金压力大

### 策略4：重邻联动

- 重号：上期号码中选1-2个
- 邻号：上期号码±1选1-2个
- 其他：补充热号或冷号

历史验证：约85%期次有重号

### 策略5：杀号排除

先杀号，再从剩余号中选。

常用杀号条件：
- 杀上期头尾号
- 杀连续3期未出号码
- 杀AC值过低组合

### 策略6：机选修正

1. 机选一注
2. 根据分析调整1-2个号码
3. 保留机选的随机性

优点：避免主观偏见

## 三、资金管理

### 定额投注

每期固定金额，如20元/期。

月支出 = 20 × 13期 = 260元
年支出 = 260 × 12 = 3120元

原则：不超过可支配收入的1-2%

### 递增投注（谨慎使用）

未中奖时逐步增加投注。

示例（保守型）：
- 第1-4期：2元
- 第5-8期：4元
- 第9-12期：8元
- 超过12期：重置

风险：资金耗尽前可能仍未中奖

### 周期投注

划分投注周期，每周期固定预算。

示例：
- 每月预算：200元
- 每期：15元（约13期）
- 超出预算则停止

### 止盈止损

**止盈**：
- 中大奖后降低投注
- 设定盈利目标

**止损**：
- 连续N期未中任何奖则暂停
- 月度亏损达上限则停止

## 四、追号策略

### 定号追号

确定一组号码，长期追投。

周期估算：
- 一等奖概率：1/17,721,088
- 每年154期
- 理论回报周期：11.5万年（单注）

结论：单注追号不现实

### 渐进追号

前N期固定，后期加倍。

问题：加倍速度跟不上概率

### 分组轮换

准备多组号码轮换投注。

示例：5组号码，每期投1组
- 成本：2元/期
- 覆盖：10元的号码量

## 五、合买策略

### 合买优势

- 降低个人成本
- 增加号码覆盖
- 分散风险

### 合买计算

示例：10人合买复式票

票面：红10蓝3 = 210×3 = 630注 = 1260元
每人：126元

中奖分配：按份额比例

### 合买风险

- 组织者信任问题
- 出票凭证保存
- 中奖后分配纠纷

建议：选择正规平台合买

## 六、心理管理

### 常见心理陷阱

1. **赌徒谬误**：认为"该出了"
2. **确认偏误**：只记住验证自己想法的信息
3. **沉没成本**：因为已投入很多而继续
4. **过度自信**：相信自己能"算出"号码

### 健康心态

- 当作娱乐消费
- 接受大概率不中
- 不影响正常生活
- 中奖是惊喜而非目标

### 风险信号

如果出现以下情况，应暂停：
- 借钱买彩票
- 影响日常开支
- 情绪因彩票大起大落
- 花大量时间研究"规律"

## 七、中奖后管理

### 小奖（千元内）

- 可继续投注
- 或提现消费

### 中奖（万元级）

- 建议提现大部分
- 少量继续娱乐

### 大奖（百万级）

1. **保密**：不要声张
2. **冷静**：给自己1-2周冷静期
3. **规划**：制定资金使用计划
4. **理财**：咨询专业理财顾问
5. **生活**：不要立即辞职或大变

### 税务说明

- 单注奖金≤1万：免个税
- 单注奖金>1万：缴纳20%个税
- 由彩票中心代扣代缴

## 八、工具使用

### 推荐工具

1. **走势图**：中彩网官方走势图
2. **遗漏查询**：各大彩票网站
3. **历史数据**：本skill数据脚本
4. **概率计算**：Excel或Python

### 不推荐

- 付费预测软件（无用）
- 所谓"内部消息"（骗局）
- 高价培训课程（忽悠）

### 记录习惯

建议记录：
- 每期投注号码
- 投注金额
- 中奖情况
- 月度/年度统计

便于复盘和控制支出
